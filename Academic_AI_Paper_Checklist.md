# Academic AI Paper Checklist

Source prompt: Master_Prompt_Academic_AI_Paper_ChinaXiv_Reproducible.docx

## Goal

Produce a publication-quality AI academic paper in ChinaXiv preprint style with reproducible source code and no fabricated results, citations, datasets, or implementation details.

## Required Paper Sections

- Title
- Authors
- Abstract
- Keywords
- Introduction
- Related Work
- Theoretical Foundation
- Methodology
- Comparative Analysis
- Discussion
- Limitations
- Conclusion
- Future Work
- References
- Code Availability

## Content Rules

- Use only real literature from top-tier venues such as NeurIPS, ICML, ICLR, CVPR, ACL, and AAAI.
- Do not invent benchmark numbers.
- If experiments cannot be executed, say so clearly and provide reproducibility instructions instead of fake results.
- Ensure every table, figure, and equation is technically correct and reproducible.
- Include implementation details such as optimizer, learning rate, epochs, hardware, software versions, and random seed.

## If This Is Original Research

Include a complete reproducible repository with:

- preprocessing
- training
- evaluation
- inference
- configs
- requirements.txt
- README.md
- repository structure

## Code Availability

- Add a placeholder GitHub URL: https://github.com/<username>/<repository>
- Replace it only after the repository actually exists.

## Final Verification

- Confirm all required paper sections are present.
- Confirm all references are genuine.
- Confirm all reported results are real and reproducible.
- Confirm the code package matches the methods described in the paper.

## Suggested Paper Skeleton

1. Title and author block
2. Abstract and keywords
3. Introduction
4. Related work
5. Theoretical foundation
6. Methodology
7. Experimental setup and comparative analysis
8. Discussion and limitations
9. Conclusion and future work
10. Code availability and references
