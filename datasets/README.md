# Dataset Notes

This survey package does not redistribute benchmark datasets. For reproducibility, dataset access should follow the official licenses and download procedures of each benchmark or source project.

Benchmarks discussed in the paper include VQA-style evaluation, multimodal instruction-following evaluation, perception/reasoning probes, hallucination checks, and expert-domain multimodal reasoning benchmarks.

Do not add private, scraped, or license-restricted data to this folder.
